# Starter Project Code

## index.html

```html
<!DOCTYPE html>
<html>
<head>
<title>Starter Site</title>
<link rel="stylesheet" href="style.css">
</head>
<body>

<h1>Welcome</h1>
<p>Edit this project into your own website.</p>

</body>
</html>
```

## style.css

```css
body{
    font-family: Arial;

}
```

## How to Run

1. Save both files in the same folder.
2. Open `index.html` in a web browser.
3. Edit the content and styling to make it your own project.
